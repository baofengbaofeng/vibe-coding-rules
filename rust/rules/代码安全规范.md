# 代码安全规范

## 概述
本文档定义Rust项目的安全开发规范，基于OWASP Top 10和Rust安全最佳实践，确保代码的安全性、可靠性和可维护性。

---

## 安全原则

### 1. 最小权限原则
- 每个模块、函数只拥有完成其任务所需的最小权限
- 使用适当的可见性修饰符（pub, pub(crate), 私有）
- 避免暴露内部实现细节

### 2. 防御性编程
- 假设所有输入都是恶意的
- 验证所有外部数据
- 处理所有可能的错误情况

### 3. 安全默认值
- 默认拒绝，需要时显式允许
- 默认使用安全配置
- 默认启用安全特性

### 4. 纵深防御
- 多层安全防护
- 不依赖单一安全机制
- 定期安全审计

## 注入防护

### 1. SQL注入防护
```
// 危险：字符串拼接SQL
let query = format!("SELECT * FROM users WHERE id = {}", user_id);

// 安全：使用参数化查询（SQLx）
let user = sqlx::query_as!(
    User,
    "SELECT * FROM users WHERE id = $1",
    user_id
)
.fetch_optional(pool)
.await?;

// 安全：使用ORM（SeaORM）
let user = User::find_by_id(user_id)
    .one(pool)
    .await?;
```

### 2. 命令注入防护
```
use std::process::Command;

// 危险：使用用户输入构建命令
let output = Command::new("echo")
    .arg(user_input)  // 用户可能输入 `; rm -rf /`
    .output()?;

// 安全：使用参数列表，避免shell
let output = Command::new("echo")
    .arg("固定文本")
    .output()?;

// 安全：严格验证用户输入
if !is_safe_input(user_input) {
    return Err(Error::InvalidInput);
}
```

### 3. 路径遍历防护
```
use std::path::{Path, PathBuf};

// 危险：直接使用用户提供的路径
let file_path = Path::new("/base/dir").join(user_provided_path);

// 安全：规范化并检查路径
let mut path = PathBuf::from("/base/dir");
path.push(user_provided_path);
path = path.canonicalize()?;

// 检查是否在允许的目录内
if !path.starts_with("/base/dir") {
    return Err(Error::PathTraversal);
}
```

## 访问控制

### 1. API鉴权
```
use axum::{
    extract::{State, Request},
    middleware::Next,
    response::Response,
};
use jsonwebtoken::{decode, Validation, DecodingKey};

/// JWT认证中间件
pub async fn auth_middleware(
    State(state): State<AppState>,
    mut request: Request,
    next: Next,
) -> Result<Response, AppError> {
    // 从Header提取Token
    let token = request.headers()
        .get("Authorization")
        .and_then(|h| h.to_str().ok())
        .and_then(|s| s.strip_prefix("Bearer "))
        .ok_or(AppError::Unauthorized)?;
    
    // 验证Token
    let token_data = decode::<Claims>(
        token,
        &DecodingKey::from_secret(state.config.jwt_secret.as_ref()),
        &Validation::default(),
    ).map_err(|_| AppError::Unauthorized)?;
    
    // 将用户信息存入请求扩展
    request.extensions_mut().insert(token_data.claims);
    
    Ok(next.run(request).await)
}

/// 需要特定权限的中间件
pub async fn require_role<B>(
    request: Request<B>,
    next: Next<B>,
    required_role: Role,
) -> Result<Response, AppError> {
    let claims = request.extensions()
        .get::<Claims>()
        .ok_or(AppError::Unauthorized)?;
    
    if claims.role != required_role {
        return Err(AppError::Forbidden);
    }
    
    Ok(next.run(request).await)
}
```

### 2. 权限校验
```
/// 业务层权限检查
pub async fn update_user_profile(
    &self,
    user_id: i64,
    updater_id: i64,
    updates: UserUpdates,
) -> Result<User, ServiceError> {
    // 检查是否是自己或管理员
    let updater = self.user_repository.find_by_id(updater_id).await?;
    
    if updater_id != user_id && !updater.is_admin() {
        return Err(ServiceError::PermissionDenied);
    }
    
    // 执行更新
    let user = self.user_repository.update(user_id, updates).await?;
    
    Ok(user)
}

/// 数据级权限控制
pub async fn get_user_orders(
    &self,
    user_id: i64,
    requester_id: i64,
) -> Result<Vec<Order>, ServiceError> {
    // 用户只能查看自己的订单，管理员可以查看所有
    if requester_id != user_id {
        let requester = self.user_repository.find_by_id(requester_id).await?;
        if !requester.is_admin() {
            return Err(ServiceError::PermissionDenied);
        }
    }
    
    let orders = self.order_repository.find_by_user_id(user_id).await?;
    
    Ok(orders)
}

### 2. 文件上传验证
```
use axum::{
    extract::Multipart,
    body::Bytes,
};
use mime::Mime;

const MAX_FILE_SIZE: usize = 10 * 1024 * 1024; // 10MB
const ALLOWED_MIME_TYPES: &[&str] = &[
    "image/jpeg",
    "image/png",
    "application/pdf",
];

pub async fn upload_file(
    mut multipart: Multipart,
) -> Result<Json<UploadResponse>, AppError> {
    while let Some(field) = multipart.next_field().await? {
        let name = field.name()
            .ok_or(AppError::InvalidInput)?
            .to_string();
        
        if name == "file" {
            let data = field.bytes().await?;
            
            // 检查文件大小
            if data.len() > MAX_FILE_SIZE {
                return Err(AppError::FileTooLarge);
            }
            
            // 检查文件类型
            let content_type = field.content_type()
                .map(|s| s.parse::<Mime>())
                .transpose()?
                .ok_or(AppError::InvalidFileType)?;
            
            if !ALLOWED_MIME_TYPES.contains(&content_type.as_ref()) {
                return Err(AppError::InvalidFileType);
            }
            
            // 保存文件
            let file_path = save_file(&data).await?;
            
            return Ok(Json(UploadResponse { file_path }));
        }
    }
    
    Err(AppError::MissingFile)
}
```

pub fn sanitize_filename(filename: &str) -> Result<String, Error> {

        .replace("\\", "")

    }
    
    // 检查长度
    if sanitized.len() > 255 {
        return Err(Error::FilenameTooLong);
    }
    
    Ok(sanitized)
}

/// 重定向HTTP到HTTPS
pub async fn redirect_https_middleware(
    uri: Uri,
) -> Result<Response, AppError> {
    let mut parts = uri.into_parts();
    
    // 修改scheme为https
    parts.scheme = Some(http::uri::Scheme::HTTPS);
    
    // 重建URI
    let https_uri = Uri::from_parts(parts)
        .map_err(|_| AppError::InternalError)?;
    
    // 返回重定向
    Ok((
        StatusCode::PERMANENT_REDIRECT,
        [(header::LOCATION, https_uri.to_string())],
    ).into_response())
}

use argon2::{

use rand_core::OsRng;

/// 哈希密码
pub fn hash_password(password: &str) -> Result<String, PasswordHasherError> {
    let salt = SaltString::generate(&mut OsRng);
    let argon2 = Argon2::default();
    
    let password_hash = argon2.hash_password(password.as_bytes(), &salt)?;
    
    Ok(password_hash.to_string())
}

/// 验证密码
pub fn verify_password(password: &str, hash: &str) -> Result<bool, PasswordHasherError> {
    let parsed_hash = PasswordHash::new(hash)?;
    let argon2 = Argon2::default();
    
    Ok(argon2.verify_password(password.as_bytes(), &parsed_hash).is_ok())
}
```

### 2. JWT令牌
```
use jsonwebtoken::{encode, decode, Header, Validation, EncodingKey, DecodingKey};
use serde::{Deserialize, Serialize};
use chrono::{Utc, Duration};

#[derive(Debug, Serialize, Deserialize)]
pub struct Claims {
    pub sub: String,  // 用户ID
    pub exp: usize,   // 过期时间
    pub iat: usize,   // 签发时间
    pub role: String, // 用户角色
}

/// 生成JWT令牌
pub fn generate_token(user_id: &str, role: &str) -> Result<String, Error> {
    let expiration = Utc::now()
        .checked_add_signed(Duration::hours(24))
        .expect("valid timestamp")
        .timestamp();
    
    let claims = Claims {
        sub: user_id.to_owned(),
        exp: expiration as usize,
        iat: Utc::now().timestamp() as usize,
        role: role.to_owned(),
    };
    
    let token = encode(
        &Header::default(),
        &claims,
        &EncodingKey::from_secret(env::var("JWT_SECRET")?.as_ref()),
    )?;
    
    Ok(token)
}

/// 验证JWT令牌
pub fn verify_token(token: &str) -> Result<Claims, Error> {
    let token_data = decode::<Claims>(
        token,
        &DecodingKey::from_secret(env::var("JWT_SECRET")?.as_ref()),
        &Validation::default(),
    )?;
    
    Ok(token_data.claims)
}
```

### 3. 数据加密
```
use aes_gcm::{
    aead::{Aead, KeyInit, OsRng},
    Aes256Gcm, Nonce,
};
use base64::{engine::general_purpose, Engine as _};

/// 加密数据
pub fn encrypt_data(plaintext: &str, key: &[u8; 32]) -> Result<String, Error> {
    let cipher = Aes256Gcm::new(key.into());
    let nonce = Nonce::from_slice(b"unique nonce"); // 实际应用中应该使用随机nonce
    
    let ciphertext = cipher.encrypt(nonce, plaintext.as_bytes())
        .map_err(|_| Error::EncryptionFailed)?;
    
    Ok(general_purpose::STANDARD.encode(ciphertext))
}

/// 解密数据
pub fn decrypt_data(ciphertext: &str, key: &[u8; 32]) -> Result<String, Error> {
    let cipher = Aes256Gcm::new(key.into());
    let nonce = Nonce::from_slice(b"unique nonce");
    
    let data = general_purpose::STANDARD.decode(ciphertext)?;
    let plaintext = cipher.decrypt(nonce, data.as_ref())
        .map_err(|_| Error::DecryptionFailed)?;
    
    String::from_utf8(plaintext).map_err(|_| Error::DecryptionFailed)
}
```

### 1. 安全事件处理流程

/// 安全事件处理
pub async fn handle_security_incident(
    incident_type: IncidentType,
    severity: Severity,
    details: &str,
) -> Result<(), Error> {
    // 1. 记录事件
    log_security_event("incident", details, None);
    
    // 2. 根据严重程度采取行动
    match severity {
        Severity::Low => {
            // 记录日志，无需立即行动
            info!("低风险安全事件: {}", details);
        }
        Severity::Medium => {
            // 通知相关人员
            notify_security_team(incident_type, details).await?;
        }
        Severity::High => {
            // 立即采取行动
            take_immediate_action(incident_type).await?;
            notify_security_team(incident_type, details).await?;
            escalate_to_management().await?;
        }
        Severity::Critical => {
            // 紧急响应
            trigger_emergency_protocol().await?;
            notify_all_stakeholders().await?;
        }
    }
    
    // 3. 后续跟进
    schedule_follow_up(incident_type).await?;
    
    Ok(())
}

/// 密钥轮换策略
pub struct KeyRotation {
    current_key: Arc<RwLock<String>>,
    previous_key: Arc<RwLock<Option<String>>>,
    rotation_interval: Duration,
}

impl KeyRotation {
    pub async fn rotate_key(&self) -> Result<(), Error> {
        // 生成新密钥
        let new_key = generate_secure_key();
        
        // 更新密钥
        let mut current = self.current_key.write().await;
        let mut previous = self.previous_key.write().await;
        
        *previous = Some(current.clone());
        *current = new_key;
        
        // 记录密钥轮换
        info!("密钥已轮换");
        
        Ok(())
    }
    
    pub async fn verify_token_with_key_rotation(
        &self,
        token: &str,
    ) -> Result<Claims, Error> {
        // 尝试用当前密钥验证
        match verify_token_with_key(token, &self.current_key.read().await) {
            Ok(claims) => return Ok(claims),
            Err(_) => {
                // 如果当前密钥失败，尝试用旧密钥
                if let Some(prev_key) = &*self.previous_key.read().await {
                    return verify_token_with_key(token, prev_key);
                }
                return Err(Error::InvalidToken);
            }
        }
    }
}

2. **cargo-deny**: 许可证和依赖检查
3. **clippy**: 代码安全警告
4. **rustsec**: Rust安全数据库
5. **sccache**: 构建缓存安全
